# ExFreight West - Franchise Website

This is a licensed authorized franchise website clone of ExFreight, customized for the West Coast location.

## Features

- Full clone of ExFreight service pages
- All trucking, air freight, ocean freight, and shipping services
- How-to-ship guides and videos
- Automatic deployment to SiteGround via GitHub Actions

## Deployment

This repository is configured for automatic deployment to SiteGround. Every push to `main` triggers a deployment.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
